# Silent Study Partner

Silent Study Partner is a Flask web app that helps students find study partners based on shared availability.

## Features
- Add availability
- Find overlapping study times
- Send and manage invitations

## Tech
- Python (Flask)
- JSON
- HTML

## Run the app

pip install flask  
python3 app.py   # Mac/Linux  
python app.py    # Windows  

Then open: http://127.0.0.1:5000
